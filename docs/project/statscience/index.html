Objetivo

O projeto do pacote statscience está em fase de desenvolvimento, e tem por objetivo de apresentar as principais técnicas de análises de dados na área da Estatística. Seguem alguns dos tópicos abordados:

  • Estatística Experimental;
  • Procedimento de Comparações múltiplas;
  • Modelos de Regressão linear e Não Linear;
  • Modelos Lineares Generalizado;
  • Estatística Multivariada;
  • Estatística Espacial;
  • Análise Bayesiana;
  • Análise exploratória de dados;
  • Dentre outros temas

O projeto está baseado no ambiente R de programação, isto é, toda a programação na análise de dados. Contudo, queremos deixar o pacote intuitivo para não programadores, e sim, usuários da ferramenta estatistica. Para isso, criaremos duas interfaces gráficas ao usuário, uma baseada na linguagem Tcl/Tk e outra via web (usando o React). Para esse último, a ideia é tornar o usuário totalmente independente de instalação e sistema operacional, do qual só será necessário ter internet e acessar o link do projeto. A escolha da linguagem Tcl/Tk é devido a integramação com o R, uma vez que já existe o pacote tcltk integrada na base de instalação do prórpio ambiente.

Parcerias

Estamos buscando parcerias, sempre com a finalidade de propagação de conhecimento. Queremos auxiliar a ciência de um modo geral. Portanto, esse projeto é de código aberto e qualquer um poderá contribuir. Para fazer parte desse projeto, entre em contato por email, descrevendo a área em que poderá nos auxiliar. Será um imenso prazer compartilharmos as nossas experiências nesse projeto.
