Apresentação

Material desenvolvido para que os alunos da disciplina Linguagem R na Ciência de Dados (LRCD) possam desenvolver seus relatórios estatísticos ou dinâmicos para a disciplina.

O que é necessário?

  • R e RStudio
  • $\LaTeX$
  • Quarto
  • Github
  • LEEM

De forma mais detalhada recomendamos acessar este link.

Como aprender cada uma das ferramentas necessárias?

  • Para aprender sobre o R e RStudio, recomendamos:

    • curso EAR - R Básico
    • R Básico
    • R Básico 2024
    • R Básico 2023
    • R Básico 2021
    • R Intermediário 2021

  • Para aprender sobre o $\LaTeX$, recomendamos:

    • Curso LaTeX
    • LaTeX: Um guia prático com aplicações na pesquisa e educação

  • Para aprender o Quarto, recomendamos acessar a sua própria página https://quarto.org/. Como alternativa, temos um curso usando o Quarto para a criação de livros, que pode auxiliar no seu uso, como também o Curso Documentações em R. Acesse: Quarto livros e Documentações em R

  • Para aprender a usar o Github, recomendamos fortemente enteder a noção básica sobre o Git. Acesse o manual

  • Para aprender o LEEM, acesse:

    • Use leem
    • Curso do pacote leem
